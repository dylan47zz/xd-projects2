# 企业转型规划编制大纲及资料清单（民生银行内蒙古分行案例适用版）

## 一、编制说明
本大纲用于支持民生银行内蒙古分行在转型金融业务中识别目标企业、组织资料收集并编制企业转型规划。大纲参考河北省钢铁企业转型规划框架，并结合内蒙古转型金融标准试用场景进行通用化调整，可适用于煤电、钢铁、建材、化工、有色等具备明确转型路径的重点行业企业。

本大纲的主要用途包括：
1. 作为银行与企业开展前期沟通的提纲；
2. 作为企业内部组织经营、技术、财务、环保等部门准备材料的依据；
3. 作为后续编制企业转型规划正文和设计融资服务方案的基础；
4. 作为银行识别企业是否符合转型金融支持条件的重要参考。

## 二、企业转型规划建议目录

### 1. 企业基本情况
1.1 企业概况
- 企业名称、统一社会信用代码、注册地址、生产经营地址
- 法定代表人、联系人及联系方式
- 成立时间、股权结构、员工人数、主要子公司或分支机构情况

1.2 行业与业务情况
- 所属行业及国民经济行业分类代码
- 主营业务、主要产品及产能情况
- 主要生产工艺和业务流程
- 上下游产业链情况及区域市场地位

1.3 经营与信用情况
- 近三年营业收入、利润、资产负债等基本财务情况
- 近三年主要融资情况及存量贷款情况
- 近三年信用记录、环保处罚、行政处罚及合规情况说明

1.4 能耗与碳排放基础
- 近两年至少一个完整年度的能源消费情况（煤、电、气、热、油等）
- 近两年至少一个完整年度的温室气体排放情况
- 单位产值碳排放强度或单位产品碳排放强度情况
- 如已开展碳核算、碳盘查、能效审计、节能评估等，请说明成果情况

### 2. 转型基础与适配性判断
2.1 政策与标准适配性
- 企业所属行业是否纳入转型金融标准试用范围
- 拟支持项目是否符合行业转型方向与技术路径
- 企业是否具备开展转型金融业务的基本资料与管理基础

2.2 企业转型基础
- 当前生产技术水平与主要高耗能/高排放环节
- 既有节能降碳工作基础
- 已实施或拟实施的绿色低碳改造项目
- 企业管理层对低碳转型的重视程度及组织基础

2.3 主要问题与转型必要性
- 当前能耗、排放、设备、工艺或产品结构方面的主要问题
- 面临的政策约束、市场压力、成本压力或客户要求
- 实施转型的必要性与紧迫性

### 3. 转型战略与目标
3.1 转型总体思路
- 围绕国家“双碳”目标、行业低碳发展方向和企业经营战略，明确总体转型方向

3.2 分阶段转型目标
- 短期目标（建议至2025年或未来1—2年）
- 中期目标（建议至2030年）
- 长期目标（建议与碳中和远景衔接）

3.3 目标设置建议
- 碳强度下降目标
- 综合能耗下降目标
- 能效提升目标
- 清洁能源替代目标
- 关键工艺升级或产品结构优化目标

### 4. 未来3—5年转型项目与降碳计划
4.1 项目总体安排
- 按年度列示未来3—5年重点转型项目
- 明确项目启动时间、建设周期和实施顺序

4.2 单个项目信息建议包括
- 项目名称
- 项目建设内容
- 对应转型技术或实施路径
- 关联工序或生产环节
- 投资金额
- 预计节能降碳效果
- 项目实施前后主要指标变化
- 项目当前进展与计划安排

4.3 整体降碳路径说明
- 说明各项目之间的逻辑关系
- 说明项目实施后对企业整体碳强度、能耗和经营效率的改善作用

### 5. 财务测算与融资需求
5.1 转型投资估算
- 未来3—5年转型项目总投资
- 分年度投资安排
- 已投入金额和待投入金额

5.2 资金来源安排
- 自有资金安排
- 银行融资需求
- 其他外部融资安排（如有）

5.3 融资需求设计
- 意向融资金额
- 融资期限建议
- 资金用途说明
- 预计还款来源
- 与项目实施节奏的匹配关系

### 6. 转型计划实施的保障措施
6.1 组织保障
- 是否设置转型计划牵头部门
- 是否建立跨部门协同机制
- 管理层是否参与统筹推动

6.2 人员与能力建设
- 是否配置绿色低碳相关专业人员
- 是否开展员工培训、技术培训或数据管理培训

6.3 监督管理与绩效考核
- 是否建立转型目标、碳排放或关键绩效指标监测机制
- 是否将转型目标纳入相关部门或岗位考核

6.4 风险识别与管控
- 政策风险
- 市场风险
- 技术风险
- 经营与资金风险
- 相应防范措施

### 7. 公正转型与社会责任
7.1 对员工和就业的影响
- 转型计划是否会影响员工人数、岗位结构和收入水平

7.2 对供应链和区域民生的影响
- 是否会对上下游企业、区域供应稳定或价格产生明显影响

7.3 无重大损害原则
- 转型项目是否可能造成新的环境污染、生态破坏或其他可持续发展负面影响
- 如存在潜在影响，拟采取何种缓释措施

### 8. 信息披露与承诺责任
8.1 信息披露安排
- 转型计划落实情况
- 阶段性目标进展情况
- 转型融资资金使用情况
- 关键绩效指标完成情况
- 公正转型相关情况

8.2 披露方式
- 向银行定期提供
- 授权相关平台采集并供银行查阅
- 其他双方认可的方式

8.3 企业承诺
- 所提供资料真实、准确、完整
- 近三年信用与合规情况良好
- 愿意按银行要求持续配合信息报送、进展跟踪和效果评估

## 三、资料收集清单

为便于编制企业转型规划，建议银行向目标企业收集以下资料：

### （一）基础资料
1. 企业营业执照、组织架构图、主要子公司情况
2. 企业简介、主要产品介绍、生产工艺说明
3. 厂区分布、主要生产装置或生产线清单
4. 近三年审计报告或财务报表
5. 近三年主要融资与授信情况说明

### （二）生产经营与技术资料
1. 近两年至近三年的产量、产值、主要原材料消耗数据
2. 主要生产工艺流程图及关键设备清单
3. 现有节能、环保、技改项目资料
4. 技术改造、设备更新、能源替代等拟实施项目立项或内部方案
5. 能评、环评、节能验收、环保验收等相关文件（如有）

### （三）能源与碳排放资料
1. 近两年至少一个完整年度的煤、电、气、热、油等能源消耗数据
2. 电费单、燃料采购凭证、能源统计台账等支撑材料
3. 已开展碳核算、碳盘查、温室气体排放报告的相关材料
4. 单位产品综合能耗、单位产值能耗或碳排强度相关数据
5. 节能降碳目标、历史完成情况及相关管理制度

### （四）转型项目与投资资料
1. 拟纳入转型规划的项目清单
2. 各项目建设内容、投资估算、建设周期和预期效益
3. 各项目预计节能量、减排量或能效提升效果
4. 项目已完成的审批、备案或内部决策材料（如有）
5. 项目资金筹措方案、自有资金安排及融资需求说明

### （五）管理与制度资料
1. 环境管理制度、节能管理制度、碳排放管理制度
2. 转型计划相关组织分工说明
3. 风险管理、绩效考核、信息披露等制度材料
4. 员工培训、技术研发或外部合作情况说明

### （六）合规与社会责任资料
1. 近三年行政处罚、环保处罚、信用记录说明
2. 安全生产、环保合规、排污许可等相关材料
3. 如涉及员工安置、供应链变化、区域民生影响等事项，请提供专项说明

## 四、规划编制方法建议
1. **坚持真实、审慎、可执行原则。** 对于企业尚未掌握或无法验证的数据，不宜虚构，应明确为“待补充”或“待测算”。
2. **坚持标准导向。** 应对照内蒙古转型金融试用标准逐项核验企业和项目的适配性，不宜简单以“高碳行业”替代“符合转型标准”。
3. **坚持融资衔接。** 规划内容应与企业真实项目、真实投资和真实资金用途相对应，便于后续与银行授信逻辑相匹配。
4. **坚持动态更新。** 企业转型规划不是一次性文件，可根据政策变化、项目实施进度和经营情况进行滚动更新。

## 五、银行使用建议
1. 在项目前期，用本大纲与资料清单对企业进行沟通和摸底；
2. 在筛选阶段，重点判断企业是否具备明确转型项目、较好的资料基础和可落地融资场景；
3. 在撰写阶段，由银行、企业和服务机构共同确认关键数据、关键目标和关键项目；
4. 在融资落地后，结合规划中的阶段性目标与信息披露安排，开展持续跟踪与管理。
