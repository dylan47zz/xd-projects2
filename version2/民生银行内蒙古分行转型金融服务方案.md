# 民生银行内蒙古分行转型金融服务方案

## 一、项目背景

### （一）政策与市场背景
在我国“双碳”目标持续推进的背景下，转型金融已成为绿色金融体系的重要组成部分。中国人民银行、金融监管总局、国务院办公厅等部门近两年密集出台政策，持续强调要通过金融手段支持高碳行业和高碳主体有序实施绿色低碳转型，推动绿色金融与转型金融有效衔接，提升金融资源配置对实体经济绿色转型的支撑能力。

当前，人民银行正牵头推进煤电、钢铁、建材、农业等重点行业转型金融标准研究，并在部分地区组织试用。转型金融标准的逐步落地，使商业银行在支持高碳行业客户转型时，有了更加明确的识别依据、业务边界和管理要求。与此同时，监管考核与信息披露规则也在不断强化。绿色金融评价体系中已增加转型金融及转型规划相关考核要求，企业和金融机构围绕气候转型计划、资金用途、转型绩效等方面的信息披露要求也日益明确。

### （二）内蒙古地区的业务机遇
内蒙古自治区能源资源型产业占比较高，煤电、钢铁、建材、化工、有色等行业集聚度较高，是转型金融的重要应用场景。当前内蒙古正在试用转型金融标准，这意味着当地金融机构具备先行探索、先形成案例、先积累方法的窗口期。对分行而言，若能率先建立“标准识别—企业筛选—规划编制—融资落地—持续跟踪”的工作机制，不仅有助于提升绿色金融与转型金融业务能力，也有助于抢占重点行业客户的综合金融服务先机。

### （三）民生银行内蒙古分行面临的现实需求
对于民生银行内蒙古分行而言，推动转型金融业务落地，当前主要存在以下几个关键难点：

1. **缺少明确的企业识别抓手。** 虽然区域内具备转型需求的企业数量较多，但并非所有高碳企业都符合转型金融支持条件。分行需要结合内蒙古试用标准、行业属性、企业转型基础和融资场景，对目标企业进行筛选。
2. **缺少可用于融资支持的企业转型规划文本。** 企业转型金融融资通常需要具备较为完整的转型目标、路径、项目安排、资金需求、风险管控及信息披露安排。多数企业尚未形成规范化的转型规划，导致银行难以完成尽调、授信论证和合规支撑。
3. **缺少从规划到案例落地的服务闭环。** 分行不仅需要一份研究报告，更需要一套能够在真实客户中落地应用的工作方法，包括资料收集、规划编制、融资结构设计、披露要求衔接及贷后跟踪建议。

因此，有必要围绕内蒙古转型金融试用场景，为民生银行内蒙古分行设计一套聚焦单个案例落地的转型金融服务方案，帮助分行尽快形成示范案例、沉淀方法模板并支持后续复制推广。

## 二、研究目标

本项目立足国家转型金融政策导向、内蒙古地区试用标准和民生银行内蒙古分行业务需求，围绕“识别企业、编制规划、推动落地”三大核心任务开展服务，具体目标如下：

1. **建立面向分行业务落地的转型金融服务框架。** 结合内蒙古试用转型金融标准，梳理适用行业、企业准入逻辑、规划编制要求及融资落地关键环节，形成分行可操作的方法框架。
2. **协助筛选符合标准且具备融资场景的目标企业。** 结合分行现有客户基础或拟拓展客户，按照行业属性、转型意愿、项目储备、数据基础、环境合规和融资需求等维度，识别优先服务对象。
3. **协助编制企业转型规划。** 针对拟落地企业，搭建企业转型规划大纲，指导企业准备必要资料，并基于企业经营与减排实际编制形成规范的企业转型规划文本，为后续授信和产品设计提供支撑。
4. **形成可落地的单个转型金融案例。** 结合企业转型项目投资计划、资金用途、期限结构、还款来源和转型绩效指标，提出适配的融资服务建议，支撑分行推动单个案例尽快形成。
5. **沉淀分行后续可复制的方法与模板。** 通过本次项目形成企业筛选思路、资料清单、规划框架、案例落地路径等标准化成果，为后续批量复制奠定基础。

## 三、研究内容

### （一）内蒙古转型金融政策与标准适配研究
1. 梳理国家层面转型金融、绿色金融、气候信息披露等相关政策要求，明确转型金融业务的政策依据与监管导向。
2. 梳理内蒙古地区转型金融标准试用情况、重点行业特征及区域产业转型方向，识别分行可优先关注的行业领域。
3. 结合试用标准要求，提炼企业适配性判断维度，包括行业归属、技术路径、减排方向、项目属性、数据可得性、融资用途合规性等内容。
4. 形成“标准理解+业务应用”的工作思路，为分行筛选企业和推动案例落地提供政策与标准基础。

### （二）目标企业筛选与前期尽调支持
1. 结合民生银行内蒙古分行客户基础、区域重点行业分布及项目储备情况，建立目标企业筛选逻辑。
2. 从以下维度对企业进行初步研判：
   - 所属行业是否纳入试用转型金融标准或具备明确转型路径；
   - 企业是否存在节能降碳、设备升级、工艺改造、原燃料替代、清洁能源替换等转型需求；
   - 企业是否具备较好的经营连续性、信用基础与资料基础；
   - 企业是否具备明确的融资需求、资金用途和可执行项目；
   - 企业是否愿意配合开展转型规划编制与后续信息披露。
3. 协助分行形成目标企业初筛名单，并优先锁定1家具备较高可行性的企业作为单案例落地对象，同时形成备选企业名单。
4. 根据目标企业情况，提出尽调资料清单、访谈提纲和资料补充要求，提升后续规划编制效率。

### （三）企业转型规划编制服务
围绕单个案例企业，按照“现状评估—目标设定—路径设计—项目安排—融资衔接—风险保障—披露承诺”的逻辑开展规划编制服务，主要包括：

1. **企业转型基础评估。** 梳理企业主营业务、工艺流程、主要产品、生产经营情况、能耗与碳排放现状、环保与合规情况、既有降碳措施及转型基础。
2. **标准适配性判断。** 对照内蒙古试用转型金融标准，分析企业所属行业、技术改造方向和项目属性是否具备转型金融支持基础，明确可支持边界。
3. **转型战略与目标设置。** 协助企业确定短期、中期和中长期转型目标，重点体现碳强度下降、能效提升、工艺升级、能源结构优化等方向。
4. **未来3—5年转型项目与降碳计划。** 梳理企业拟实施的重点转型项目，明确项目内容、建设时序、投资估算、预期减排效果及对生产经营的影响。
5. **财务与融资需求测算。** 梳理企业自有资金安排、项目投资计划、融资需求规模、融资期限、资金用途及预期偿债来源，为融资服务设计提供基础。
6. **实施保障与风险管控。** 从组织机制、数据管理、绩效考核、风险识别、公正转型和信息披露等方面提出安排建议，增强规划的可执行性与可信度。
7. **形成规划文本。** 在企业资料较为完整的基础上，形成《企业转型规划》初稿，并根据分行和企业反馈进行修改完善。

### （四）单个案例融资服务落地支持
1. 结合企业项目建设节奏和资金需求，提出适配的转型金融服务建议，包括但不限于流动资金支持、中长期项目融资、技改贷款、绿色/转型类产品组合等。
2. 明确融资支持需要关注的关键要素，如资金用途、期限设置、提款安排、阶段性绩效指标、披露义务和贷后跟踪要求。
3. 协助分行将企业转型规划与授信审批逻辑相衔接，提升案例落地的合规性、清晰度和可操作性。
4. 对案例落地过程中可能涉及的材料准备、信息披露、沟通汇报和后续跟踪事项提出建议。

### （五）分行成果沉淀与复制推广建议
1. 形成一套适合分行后续推广应用的企业筛选思路和工作流程。
2. 形成企业转型规划编制大纲及资料清单，便于分行与客户开展前期沟通。
3. 总结单案例推进中的关键难点、关键材料和关键节点，提炼为后续复制推广的业务经验。
4. 视需要对分行相关业务人员进行一次成果汇报或交流说明，帮助理解项目成果与应用方式。

## 四、研究计划及报价

### （一）项目成果
本项目拟形成以下成果：

1. **《民生银行内蒙古分行转型金融服务方案》** 1份。系统阐述项目背景、目标、服务内容、实施路径及案例落地思路。
2. **《企业转型规划编制大纲及资料清单》** 1份。供分行与目标企业沟通、收集资料和组织规划编制使用。
3. **目标企业初筛与服务建议** 1套。根据分行提供的客户情况，对拟落地企业进行适配性判断并提出服务推进建议。
4. **单个企业转型规划文本（初稿/终稿）** 1份。在企业资料较为完整并配合访谈的前提下，完成单个案例企业转型规划撰写与修改。
5. **单个案例融资落地建议** 1份。围绕融资用途、期限安排、绩效指标、披露与跟踪要求等提出建议。

### （二）项目实施计划
项目整体服务周期预计为 **3—4周**，具体安排如下：

#### 第一阶段：项目启动与标准对标（第1周）
- 与分行沟通项目范围、拟优先关注行业和目标客户方向；
- 梳理国家政策、内蒙古试用标准及案例适用要求；
- 明确企业筛选逻辑、资料清单和沟通安排。

#### 第二阶段：企业筛选与资料收集（第1—2周）
- 对候选企业开展初步筛选和适配性研判；
- 协助分行锁定1家重点服务企业及备选企业；
- 向企业发送资料清单并组织访谈，收集经营、工艺、能耗、碳排、项目和融资等资料。

#### 第三阶段：转型规划编制与融资方案设计（第2—3周）
- 对企业转型基础进行分析，搭建转型目标与转型路径；
- 编制企业未来3—5年转型项目安排和融资需求测算框架；
- 形成企业转型规划初稿及案例融资服务建议。

#### 第四阶段：修改完善与案例落地建议（第3—4周）
- 根据分行及企业反馈对规划文本进行修改完善；
- 形成终稿及项目总结建议；
- 对分行后续推动授信审批、产品匹配、信息披露和贷后管理提出落地建议。

### （三）项目报价
本项目总报价为 **人民币1万元**。

报价内容主要覆盖：
- 政策与标准研究；
- 企业筛选与前期沟通支持；
- 企业转型规划大纲及资料清单编制；
- 单个案例企业转型规划撰写与修改；
- 单个案例融资服务落地建议。

上述报价以项目周期内的研究、访谈、撰写和一次集中修改为基础。如后续需要增加服务对象数量、显著扩大研究范围或开展额外现场深度尽调，可由双方另行协商。

## 五、对甲方配合事项的建议

为提高项目执行效率，建议民生银行内蒙古分行在项目实施过程中协调提供以下支持：

1. 明确拟优先推进的行业方向及候选企业名单；
2. 协助安排与企业管理层、财务、生产、环保或能耗管理相关人员的沟通；
3. 协助督促企业按资料清单提供基础材料；
4. 对规划初稿和融资建议及时反馈意见，便于项目在3—4周内完成。

## 六、预期价值

通过本项目，民生银行内蒙古分行可在内蒙古转型金融标准试用阶段形成“可识别、可编制、可落地、可复制”的实践成果，具体价值包括：

1. 形成一套适用于分行业务开展的转型金融工作方法；
2. 提升分行对高碳行业客户绿色低碳转型需求的识别与服务能力；
3. 通过单个案例落地，为后续同类客户批量拓展积累经验；
4. 为分行参与区域转型金融创新、提升绿色金融综合服务能力提供支撑。

## 七、结语

在内蒙古试用转型金融标准的关键阶段，民生银行内蒙古分行具备率先形成案例、率先沉淀方法、率先服务重点客户的良好条件。本项目将坚持“标准导向、企业真实、案例落地、成果可复制”的原则，协助分行尽快推动转型金融从政策理解走向业务实践，从方案研究走向客户服务落地。
